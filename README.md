## 📦 About the Project

LogiStock is a B2B inventory and logistics management platform developed as a Final Year Project (PFE).

The application connects clients, suppliers, managers, and administrators through a centralized system that simplifies:

- Inventory Management
- Product Catalog Management
- Order Processing
- Supplier Management
- Client Management
- Contract Management
- Invoice Generation
- Digital Signature Workflow
- Shipment Tracking
- Dashboard & Analytics
- Authentication & Authorization

---

## 🎯 Objectives

The main objective of LogiStock is to digitalize and optimize the supply chain process by providing a modern and secure platform for managing inventory and logistics operations.

# 🚀 Installation

## Clone Repository

```bash
git clone [https://github.com/LogiStock-Team/LogiStock_PFE_2026.git]
```

## Backend

```bash
cd backend

mvn clean install

mvn spring-boot:run
```

Backend URL:

```text
http://localhost:8080
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend URL:

```text
http://localhost:5173
```
